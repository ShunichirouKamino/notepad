## extensions

- インストールしたい拡張機能パッケージページへ移動
- コマンドパレットオープン(F1 or Ctrl + Shift + P)
- `Extensions: Add Extension to Workspace Folder Recomendations`を選択
- 自動で`./vscode/extensions.json`が生成される

## shortcut

- `` Ctrl + ` ``
  - ターミナルのオープン／クローズ
- Shift + Alt + 選択
  - 矩形選択
- `Ctrl + Shift + K`
  - 行削除
- `Ctrl + Space`
  - オートコンプリート類推
- 行選択 + `Ctrl + /`
  - コメントアウト／解除
